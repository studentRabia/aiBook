# Welcome to Physical AI & Humanoid Robotics

## About This Textbook

This interactive textbook provides comprehensive, hands-on coverage of Physical AI and Humanoid Robotics, from ROS 2 fundamentals to deploying Vision-Language-Action (VLA) models on humanoid systems.

### What You'll Learn

- **ROS 2 Fundamentals**: Nodes, topics, services, Python-ROS bridge, URDF modeling, and controller development
- **Digital Twin Development**: Building photorealistic simulations with Gazebo and Unity
- **NVIDIA Isaac Integration**: Synthetic data generation, hardware-accelerated perception, and bipedal navigation
- **Vision-Language-Action (VLA)**: Multimodal learning for natural language robot control
- **Complete System Integration**: End-to-end humanoid system with pickup-and-place capabilities

## Prerequisites

Before starting this textbook, you should have:

- **Programming**: Intermediate Python skills (functions, classes, async/await)
- **AI/ML**: Foundational knowledge (neural networks, reinforcement learning basics)
